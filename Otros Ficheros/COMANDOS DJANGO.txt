- Crear aplicacion: python manage.py startapp NOMBREAPP
- Hacer migraciones 1: python manage.py makemigrations 
- Hacer migraciones 2: python manage.py migrate
- Crear superusuario: python manage.py createsuperuser
- Colectar todos los archivos estaticos: python manage.py collectstatic
- En bash: service redis-server start
- En cmd: python manage.py runserver
- En cmd: celery -A NOMBREPROYECTO.celery worker -l info -P eventlet
